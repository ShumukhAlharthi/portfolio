Projects

OptiScan - AI Medical System

OptiScan is an AI-powered eye disease detection system developed as a graduation project using Optical Coherence Tomography (OCT) images. The model was trained on approximately 84,000 retinal OCT images to classify CNV, DME, Drusen, and Normal cases using deep learning models including CNN, VGG16, ResNet50, and DenseNet121. The project achieved 98% accuracy and won third place in the graduation projects competition at Taif University.
